Kudosu

Kudosu is a random sudoku generator that allows sudoku lovers an endless stream of sudoku puzzles. This is my most recent project, and is still set to receive some major updates in the coming months. The main focus of this project was for experimentation with advanced data structures and algorithms. The sudoku generation uses Donald Knuth's Dancing Links algorithm and requires the construction of a circularly-linked sparse matrix.

Stack: JavaScript, Python, Django, PostgreSQL

Online Tools: GitHub, Heroku, Figma

Languages: HTML, CSS, JavaScript, Python
